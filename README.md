# mailchimp

# A Mailchimp API implementation in Dart


## Basic Usage:

```dart
import 'package:mailchimp/mailchimp.dart';
```

## MailChimp Marketing:

### Get Root

```dart
  MailChimp mailChimp = MailChimp.marketing(apiKey: "sdf", server: "Sdf");
  mailChimp.getRoot();
```
